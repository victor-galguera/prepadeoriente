<h2>Post installation configuration instructions.</h2>

<iframe width="420" height="315" src="https://www.youtube.com/embed/nqlKwgVnQuw"
frameborder="0" allowfullscreen></iframe>

<ol>
<li>Visit <a href="/admin/content/youtube-import">
yoursite.com/admin/content/youtube-import</a> and review the options.</li>
<li>Get an <strong>API Key</strong> and set it in "YouTube API key".
<ol>

<li>To get an API key go to 

<a href="https://console.developers.google.com/apis/library">
https://console.developers.google.com/apis/library</a></li>

<li>If you haven't already created one create a new project.
 Click "<strong>Select A Project...</strong>" then 
"<strong>Create a project</strong>". If you have already created a project
 "Select a project" will be the project name.</li>

<li>In the search box type "Youtube" and then click on "YouTube Data API v3"
</li>

<li>Click on "<strong>Enable</strong>"  and then on 
"<strong>Credentials</strong>"</li>

<li>Click "Create Credentials" <strong>--></strong> "API Key" 
<strong>--></strong> "Server Key"</li>

<li>Give it any name thats relevant to you and then under "Accept requests from
 these server IP addresses" enter the IP address of your website.
<li>

<li>If you don't know your IP address open a command prompt and ping your
 website. It will give you the IP address.
</li>

<li>Click "<strong>Create</strong>" and copy the API key you are given.</li>
</ol></li>
<li>Type your youtube user name or channel ID.</li>

<li>You can leave the playlist box blank and the module will automatically set
 it to the "<strong>All uploads</strong>" playlist. If you want to use another
 playlist visit the playlist you wish to use and look fot the playlist ID in
 the URL it looks like this "<strong>PLwesP0W1F9e6lsfZbG53IsG1pCeN-NkIY
</strong>".</li>

<li>By default the import runs automatically every hour, <strong>3600</strong>
 seconds. Set it to 0 to only run manually or specify another interval in
 seconds.
</li>

<li>Select an appropriate content type for the import. It is highly recommended 
to use the <strong>Youtube Field</strong> module, or similar in your content
 type.
</li>

<li>Set the author for the content, default is Anonymous.</li>

<li><strong>Save the settings</strong> and the options for field mapping will
 appear below.
</li>

<li>Review the Fields ensuring the Youtube data is mapped to the correct content
 fields ensuring to <strong>save again</strong> if you make any changes.</li>

<li>Click on "<strong>Click here to run the import now</strong>"</li>
</ol>
